RapidReport/java-pdf2
==================
RapidReport/java-pdf2 は Java 用の帳票レンダリングエンジンで使用する
PDF出力にiText5系を用いています。

RapidReport  
[http://rapidreport.systembase.co.jp/](http://rapidreport.systembase.co.jp/)

動作環境
-------
Java6 以上（Eclipse プロジェクト）

ライセンス
-------
AGPLライセンスに準拠します。
